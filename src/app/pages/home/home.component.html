Tutorial de Angular

¿Para qué sirve Angular?

Angular es un framework para el desarrollo de aplicaciones web modernas. Su propósito principal es:

  • Crear aplicaciones web dinámicas y de una sola página (SPA, Single Page Applications).
  • Facilitar la construcción de interfaces de usuario interactivas.
  • Proveer una arquitectura modular basada en componentes reutilizables.
  • Gestionar la interacción entre el frontend y servicios backend.

Actualmente es uno de los frameworks más populares y utilizados en el desarrollo web, gracias a su potencia y flexibilidad.

¿Qué aplicaciones tiene?

Angular se utiliza en una variedad de aplicaciones web, incluyendo:

  1. Aplicaciones empresariales:
    • Sistemas de gestión empresarial (ERP, CRM).
    • Paneles de administración.
  2. Tiendas en línea:
    • Plataformas de comercio electrónico.
  3. Sistemas educativos:
    • Portales de aprendizaje interactivo.
    • Plataformas de cursos en línea.
  4. Dashboards y herramientas de análisis:
    • Visualización de datos en tiempo real.
  5. Aplicaciones móviles y de escritorio:
    • Mediante herramientas como Ionic o Electron.
  6. Portales públicos y sitios informativos:
    • Blogs dinámicos y foros.

¿Sobre qué lenguajes se apoya?

Angular utiliza y se basa en los siguientes lenguajes y tecnologías:

  1. TypeScript: Es el lenguaje principal de Angular, una extensión de JavaScript que agrega tipado estático y características avanzadas.
  2. HTML: Para definir la estructura de las vistas.
  3. CSS/SCSS: Para dar estilo a las aplicaciones.
  4. JavaScript: Angular transpila el código TypeScript a JavaScript para que sea ejecutado en navegadores.

¿Qué prerrequisitos necesita?

Para trabajar con Angular, es necesario tener conocimientos previos y cumplir ciertos requisitos técnicos:

  1. Conocimientos básicos:
    • HTML, CSS y JavaScript.
    • Programación orientada a objetos (POO) y manejo de componentes.
    • Familiaridad con el modelo de desarrollo basado en MVC (Model-View-Controller) o MVVM (Model-View-ViewModel).
  2. Herramientas necesarias:
    • Node.js: Angular requiere Node.js para instalar paquetes y ejecutar scripts con npm (Node Package Manager).
    • Angular CLI: Una herramienta de línea de comandos para crear, desarrollar y desplegar aplicaciones Angular.
  3. Editor de código:
    • Un IDE como Visual Studio Code es altamente recomendado.
  4. Navegador moderno:
    • Para probar y depurar la aplicación.
  5. Conceptos adicionales (opcionales pero útiles):
    • Conocimientos básicos sobre APIs RESTful y cómo consumirlas.
    • Manejo de sistemas de control de versiones como Git.
