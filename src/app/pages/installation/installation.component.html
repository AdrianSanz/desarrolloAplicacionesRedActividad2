Cómo instalar Angular en tu equipo local

Compatibilidad de versiones

Antes de instalar Angular tienes que asegurarte de las versiones de las herramientas que vas a utilizar son compatibles. Por ejemplo, si vas a utilizar Angular 19, necesitas tener instalado Node.js 18.19.1 o superior. Puedes consultar la página oficial de Angular para más información.

Esta web se ha generado con las siguientes versiones:

  • Node.js: 20.9.0
  • Angular CLI: 17.3.11
  • Package Manager: npm 9.6.3

Paso 1: Instalar Node.js

Angular requiere Node.js para funcionar. Sigue estos pasos:

  1. Visita el sitio oficial de Node.js.
  2. Descarga la versión LTS recomendada para tu sistema operativo y que sea compatible con la versión de Angular que quieres utilizar.
  3. Instala Node.js siguiendo las instrucciones del instalador.

Instalar Node.js también instalará npm, el gestor de paquetes de Node.js, que necesitarás para instalar Angular.

Paso 2: Instalar Angular CLI

Una vez que tengas Node.js instalado, abre una terminal y ejecuta el siguiente comando: 

npm install -g @angular/cli

Esto instalará Angular CLI de manera global en tu equipo.

Paso 3: Instalar un IDE

Es recomendable instalar un IDE como Visual Studio Code para trabajar con Angular. Puedes descargarlo desde la página oficial de Visual Studio Code.
