Primeros pasos

Paso 1: Crear un nuevo proyecto

Posicionate en la carpeta donde quieres crear tu proyecto y abre un terminal. Para crear un nuevo proyecto en Angular, ejecuta el siguiente comando: 

ng new hello-world

Elige las siguientes opciones durante la configuración:

  • ¿Quieres habilitar el enrutamiento? No (puedes añadirlo después si lo necesitas).
  • Selecciona CSS como formato de estilos (o tu preferido).

Sustituye el contenido de app.component.html con el siguiente código para que se visualice un titulo con el texto "Hello, World!"

<h1>Hello, World!</h1>

Paso 2: Crear nuevos módulos y componentes

Si quieres crear nuevos módulos dentro de tu proyecto, para estructurar las vistas, ejecuta el siguiente comando en tu terminal: 

ng g module nombre-del-modulo

Para crear componentes dentro de tu módulo ejecuta el siguiente comando en tu terminal: 

ng g component nombre-del-componente --flat -m nombre-del-modulo

Paso 3: Enlazar módulos y componentes

Para enlazar los módulos y componentes que has creado, modifica el archivo app.module.ts y añade los módulos a la lista de imports.

Para añadir las rutas modifica el archivo app.routes.ts y añade los path con los componentes a los que hace referencia.

Asegurate que en el archivo app.component.ts tienes importados RouterModule y RouterOutlet, ya que son necesarios para enlazar los componentes.

Por último, modifica el archivo app.component.html para añadir el router-outlet que mostrará los componentes en función de la ruta.

Paso 4: Levantar la aplicación en local

En tu terminal, ejecuta el siguiente comando para levantar la aplicación en local: 

npm start

Esto levantará tu aplicación en el puerto 4200 por defecto. Abre tu navegador y accede a la dirección http://localhost:4200 para ver tu aplicación en funcionamiento.
